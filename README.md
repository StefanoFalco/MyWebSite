## Technical notes

The website is intentionally lightweight:

- static HTML, CSS and JavaScript;
- GitHub Pages deployment;
- no backend;
- no database;
- no tracking scripts;
- responsive layout;
- dark/light theme support.

This keeps the portfolio simple, fast and easy to inspect.
